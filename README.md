# Pong!

This is a recreation of the classic arcade game Pong with SFML/C++, created as my first game in the [20 Games Challenge](https://20_games_challenge.gitlab.io/)!

# Goals
## Basic goals
- [x] Arena with 2 walls + divider (01/07/26)
- [ ] User-controllable paddles
- [ ] Ball that moves around play field + bounces off paddles/walls
- [ ] Detect when ball leaves play field
- [ ] Track and display score for each player

## Stretch goals
- [ ] CPU panel to allow singleplayer
- [ ] Menu/reset option
- [ ] Play sounds when ball collides/when player scores
